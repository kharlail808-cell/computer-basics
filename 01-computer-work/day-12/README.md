# Materi

## Apa itu GPU?

GPU (Graphics Processing Unit) adalah komponen komputer yang bertugas mengolah grafis, gambar, video, dan tampilan sebelum ditampilkan ke monitor.

GPU dirancang untuk menangani jutaan piksel secara bersamaan sehingga sangat cocok digunakan untuk game, desain grafis, animasi, dan editing video.

---

## Fungsi GPU

- Mengolah grafis.
- Menampilkan gambar ke monitor.
- Membantu proses rendering video dan animasi.
- Mengurangi beban CPU pada pekerjaan grafis.

---

## CPU vs GPU

| CPU | GPU |
|------|------|
| Menjalankan instruksi | Mengolah grafis |
| Mengatur logika program | Menggambar tampilan |
| Core lebih sedikit tetapi lebih kompleks | Memiliki banyak core kecil untuk pekerjaan paralel |

---

## Jenis GPU

### Integrated Graphics (iGPU)

GPU yang menyatu dengan CPU.

Contoh:
- AMD Radeon Graphics (Ryzen Series)
- Intel UHD Graphics
- Intel Iris Xe

Kelebihan:
- Hemat daya.
- Harga lebih murah.
- Cocok untuk Office, belajar, browsing, dan multimedia.

---

### Dedicated Graphics (dGPU)

GPU yang terpisah dari CPU dan dipasang pada motherboard melalui slot PCI Express (PCIe).

Contoh:
- NVIDIA GeForce RTX
- AMD Radeon RX

Kelebihan:
- Performa grafis jauh lebih tinggi.
- Cocok untuk gaming, editing video, desain 3D, dan AI.

# Kesimpulan

GPU merupakan komponen yang bertanggung jawab dalam pengolahan grafis.

CPU menjalankan instruksi dan logika program, sedangkan GPU mengolah tampilan agar dapat ditampilkan ke monitor.

Pada pekerjaan yang membutuhkan grafis tinggi seperti game dan editing video, GPU memiliki peran yang sangat penting.
