# Materi

## Apa itu Komputer?

Komputer adalah perangkat elektronik yang menerima data (input), mengolah data sesuai instruksi (process), lalu menghasilkan informasi (output).

Alur kerja dasar komputer:

Input → Process → Output

Contoh:

- Input: Menekan tombol "A" pada keyboard.
- Process: CPU memproses instruksi.
- Output: Huruf "A" muncul di layar.

---

## Komponen Utama Komputer

### CPU (Central Processing Unit)

CPU adalah otak komputer yang bertugas menjalankan instruksi dan memproses semua pekerjaan komputer.

### RAM (Random Access Memory)

RAM menyimpan data sementara yang sedang digunakan oleh komputer agar dapat diakses dengan cepat.

### Storage (SSD)

Storage digunakan untuk menyimpan data secara permanen, seperti sistem operasi, aplikasi, dan file pengguna.

### Motherboard

Motherboard menghubungkan seluruh komponen komputer sehingga dapat bekerja bersama.

---

## Cara Kerja Saat Membuka Aplikasi

1. Pengguna mengklik ikon aplikasi.
2. SSD membaca file aplikasi.
3. Data dipindahkan ke RAM.
4. CPU menjalankan instruksi aplikasi.
5. Hasil ditampilkan pada monitor.

---

# Praktik

## Tujuan

Mengamati perubahan penggunaan CPU, RAM, dan Disk saat membuka aplikasi menggunakan Task Manager.

## Alat yang Digunakan

- Laptop
- Windows
- Task Manager

## Langkah-langkah

1. Membuka Task Manager.
2. Mengamati kondisi awal CPU, RAM, dan Disk.
3. Membuka Google Chrome beserta beberapa tab.
4. Mengamati perubahan penggunaan CPU, RAM, dan Disk.
5. Mencatat hasil pengamatan.

# Analisis

Berdasarkan hasil praktik:

- CPU bekerja lebih keras saat Google Chrome dibuka karena harus menjalankan banyak instruksi.
- Penggunaan RAM meningkat karena Chrome membutuhkan memori untuk menjalankan aplikasi dan membuka beberapa tab.
- SSD digunakan untuk membaca data aplikasi sehingga aktivitas Disk meningkat saat Chrome pertama kali dibuka.
- Setelah aplikasi selesai dimuat, aktivitas Disk kembali menurun.

---

# Kesimpulan

Saya memahami bahwa saat membuka sebuah aplikasi, komputer menggunakan beberapa komponen yang bekerja secara bersamaan.

Urutan prosesnya adalah:

1. SSD membaca file aplikasi.
2. Data dipindahkan ke RAM.
3. CPU menjalankan instruksi aplikasi.
4. Hasil ditampilkan pada monitor.

Setiap komponen memiliki fungsi yang berbeda, tetapi semuanya saling bekerja sama agar aplikasi dapat berjalan dengan baik.
