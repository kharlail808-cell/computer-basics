**Tanggal:** 7 Juli 2026

---

# Tujuan Belajar

- Memahami pengertian file dan folder.
- Mengetahui fungsi folder dalam mengelompokkan data.
- Memahami konsep path (alamat file).
- Mengetahui perbedaan Copy, Cut, Paste, dan Delete.
- Memahami fungsi Recycle Bin.

---

# Materi

## File

File adalah data yang disimpan di dalam komputer.

Contoh file:

- .docx
- .pdf
- .jpg
- .png
- .exe
- .md

---

## Folder

Folder digunakan untuk mengelompokkan file agar lebih rapi dan mudah ditemukan.

Folder juga dapat berisi folder lain (subfolder).

---

## Path

Path adalah alamat lengkap suatu file atau folder.

Contoh:

C:\Users\Khra\Documents\Belajar\README.md

Keterangan:

- C: = Drive
- Users = Folder bawaan Windows
- Khra = Folder pengguna
- Documents = Folder dokumen
- Belajar = Folder yang dibuat pengguna
- README.md = File

---

## Copy

Copy digunakan untuk membuat salinan file.

File asli tetap berada di lokasi semula.

---

## Cut

Cut digunakan untuk memindahkan file ke lokasi lain.

File akan berpindah setelah dilakukan Paste.

---

## Delete

Delete digunakan untuk menghapus file.

Secara default, file akan masuk ke Recycle Bin sehingga masih dapat dipulihkan.

---

## Recycle Bin

Recycle Bin berfungsi sebagai tempat penyimpanan sementara file yang dihapus sebelum benar-benar dihapus secara permanen.

---

# Praktik

Praktik yang dilakukan:

- Membuat struktur folder.
- Membuat beberapa file teks.
- Mencoba Copy file.
- Mencoba Cut (memindahkan) file.
- Menghapus file dan mengecek Recycle Bin.
- Melakukan Restore pada file yang dihapus.
- Mengamati perubahan path setelah file dipindahkan.

---

# Dokumentasi

Folder dokumentasi berisi:

- Struktur folder
- Hasil Copy file
- Hasil Cut file
- Recycle Bin
