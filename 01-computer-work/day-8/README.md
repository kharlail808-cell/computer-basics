# Materi

## Storage

Storage adalah media penyimpanan permanen yang digunakan untuk menyimpan sistem operasi, aplikasi, dokumen, foto, video, dan file lainnya. Data pada Storage tetap tersimpan meskipun komputer dimatikan.

## HDD (Hard Disk Drive)

- Menggunakan piringan magnet yang berputar.
- Memiliki komponen mekanis.
- Kecepatan baca dan tulis lebih lambat.
- Harga lebih murah.

## SSD (Solid State Drive)

- Menggunakan chip memori.
- Tidak memiliki komponen yang bergerak.
- Kecepatan baca dan tulis jauh lebih tinggi.
- Lebih tahan terhadap guncangan dan lebih hemat daya.

---

# Praktik

## Tujuan

Mengamati aktivitas SSD ketika membuka aplikasi dan menyalin file menggunakan Task Manager.

## Alat

- Laptop
- Windows
- Task Manager
- File Explorer

## Langkah

1. Membuka Task Manager.
2. Membuka tab **Performance** lalu memilih **Disk (SSD)**.
3. Mengamati kondisi SSD sebelum membuka aplikasi.
4. Membuka Google Chrome.
5. Mengamati perubahan Active Time, Read Speed, dan Write Speed.
6. Menyalin sebuah file ke folder lain.
7. Mengamati kembali aktivitas SSD setelah proses selesai.

---

# Hasil Pengamatan

| Kondisi | Pengamatan |
|----------|------------|
| Sebelum membuka aplikasi | Aktivitas SSD rendah. |
| Saat membuka Google Chrome | Read Speed meningkat karena SSD membaca file aplikasi. |
| Saat menyalin file | Read Speed dan Write Speed meningkat karena SSD membaca dan menulis data. |
| Setelah proses selesai | Aktivitas SSD kembali rendah. |

---

# Analisis

SSD bekerja paling sibuk ketika membaca atau menulis data.

Saat membuka Google Chrome, SSD membaca file aplikasi kemudian mengirimkannya ke RAM. Setelah data berada di RAM, CPU mengambil data tersebut untuk menjalankan instruksi.

Saat menyalin file, SSD melakukan dua pekerjaan sekaligus, yaitu membaca data dari lokasi asal dan menuliskannya ke lokasi tujuan. Oleh karena itu, Read Speed dan Write Speed meningkat.

SSD tidak membuat CPU menjadi lebih cepat, tetapi SSD mengurangi waktu tunggu CPU karena data dapat dibaca lebih cepat dibandingkan HDD.

---

# Kesimpulan

Saya memahami bahwa Storage merupakan tempat penyimpanan permanen. SSD memiliki kecepatan yang jauh lebih tinggi dibandingkan HDD karena menggunakan chip memori tanpa komponen bergerak. SSD membantu mempercepat proses membuka aplikasi dan memindahkan file dengan mengurangi waktu tunggu CPU.
