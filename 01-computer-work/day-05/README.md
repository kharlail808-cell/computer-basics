# Hari 05 - Praktik Penggunaan RAM

**Tanggal:** 03 Juli 2026

---

# Tujuan

Mengamati perubahan penggunaan RAM ketika membuka aplikasi dan melihat bagaimana RAM kembali berkurang setelah aplikasi ditutup.

---

# Alat

- Laptop
- Windows
- Task Manager
- Calculator
- Google Chrome

---

# Langkah Praktik

1. Membuka Task Manager pada tab **Performance > Memory**.
2. Mengamati penggunaan RAM sebelum membuka aplikasi.
3. Membuka Calculator.
4. Membuka Google Chrome.
5. Membuka beberapa tab pada Google Chrome.
6. Mengamati perubahan penggunaan RAM.
7. Menutup seluruh aplikasi.
8. Mengamati penggunaan RAM setelah aplikasi ditutup.

---

# Hasil Pengamatan

| Kondisi | Penggunaan RAM | Pengamatan |
|---------|---------------|------------|
| Sebelum membuka aplikasi | 54% | Penggunaan RAM masih normal untuk sistem Windows. |
| Membuka Calculator | 58% | RAM bertambah sedikit karena aplikasi ringan. |
| Membuka Google Chrome | 66% | RAM meningkat karena Chrome memuat halaman dan proses tambahan. |
| Membuka beberapa tab Chrome | 83% | Penggunaan RAM meningkat cukup besar karena setiap tab membutuhkan memori. |
| Setelah semua aplikasi ditutup | 58% | RAM kembali berkurang karena data sementara aplikasi telah dibersihkan. |

---

# Analisis

Calculator hanya membutuhkan sedikit RAM sehingga perubahan tidak terlalu besar.

Google Chrome menggunakan RAM lebih banyak karena setiap tab menjalankan proses sendiri dan menyimpan data sementara agar halaman tetap cepat diakses.

Ketika aplikasi ditutup, RAM tidak menyimpan data tersebut lagi sehingga kapasitas RAM kembali tersedia.

---

# Kesimpulan

RAM merupakan tempat penyimpanan sementara yang digunakan aplikasi saat berjalan. Semakin banyak aplikasi dan tab yang dibuka, semakin besar penggunaan RAM. Setelah aplikasi ditutup, sebagian besar penggunaan RAM akan kembali berkurang sehingga dapat digunakan oleh aplikasi lain.
