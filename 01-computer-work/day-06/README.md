# Hari 06 - Mengenal Spesifikasi RAM

Tanggal : 04 Juli 2026

## Tujuan Belajar

- Memahami spesifikasi RAM pada komputer.
- Mengetahui fungsi kapasitas, kecepatan, dan jenis RAM.
- Memahami pengaruh Single Channel dan Dual Channel terhadap performa.

---

## Materi

- Jenis RAM (DDR3, DDR4, DDR5)
- Kapasitas RAM (GB)
- Kecepatan RAM (MHz)
- Single Channel
- Dual Channel

---

## Praktik

### Tujuan

Mengidentifikasi spesifikasi RAM pada laptop menggunakan Task Manager dan System Information.

### Alat

- Laptop Lenovo
- Windows 10 Pro
- Task Manager
- System Information (msinfo32)

### Langkah

1. Membuka Task Manager.
2. Memilih tab Performance.
3. Membuka menu Memory.
4. Mencatat kapasitas RAM, jenis RAM, kecepatan, dan jumlah slot.
5. Membuka System Information menggunakan perintah `msinfo32`.
6. Mencatat model laptop dan spesifikasi utama.

## Analisis

Laptop menggunakan RAM DDR3 berkapasitas 6 GB dengan kecepatan 1333 MHz. Kedua slot RAM telah terisi sehingga konfigurasi RAM menggunakan dua keping.

Berdasarkan materi yang dipelajari, konfigurasi Dual Channel membantu CPU mengambil data dari RAM lebih cepat dibanding Single Channel.

Meskipun laptop ini masih mampu menjalankan aktivitas seperti mengetik, browsing, dan menonton video, spesifikasi prosesornya sudah cukup lama sehingga performanya tidak secepat laptop modern.
